Limpador de ROMs

Feito em Python, o software faz uma varredura na pasta selecionada e, pelo hash dos arquivos, consegue identificar se há arquivos repetidos, dando a opção do usuário selecionar os itens que deseja excluir do diretório.

O software ainda oferece opções de extensões para serem selecionadas para varreduras mais precisas.

Obs.: Projeto realizado com o auxílio de IA.
